# @macalinao/clients-token-metadata

TypeScript client for the Metaplex Token Metadata program

## Modules

- [Instructions](./token-metadata/instructions) - Transaction instruction builders
- [Accounts](./token-metadata/accounts) - Account decoders and fetchers
- [Types](./token-metadata/types) - TypeScript type definitions
- [PDAs](./token-metadata/pdas) - Program-derived address helpers
- [Errors](./token-metadata/errors) - Error enums and handlers
- [Programs](./token-metadata/programs) - Program IDs and utilities

## Installation

```bash
bun add @macalinao/clients-token-metadata
```

## Usage

```typescript
import {} from /* exports */ "@macalinao/clients-token-metadata";
```
