# @macalinao/clients-quarry

TypeScript client for Quarry protocol programs

## Modules

- [Instructions](./quarry/instructions) - Transaction instruction builders
- [Accounts](./quarry/accounts) - Account decoders and fetchers
- [Types](./quarry/types) - TypeScript type definitions
- [PDAs](./quarry/pdas) - Program-derived address helpers
- [Errors](./quarry/errors) - Error enums and handlers
- [Programs](./quarry/programs) - Program IDs and utilities

## Installation

```bash
bun add @macalinao/clients-quarry
```

## Usage

```typescript
import {} from /* exports */ "@macalinao/clients-quarry";
```
