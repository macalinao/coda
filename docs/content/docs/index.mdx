# Welcome to Coda

Coda is an automated client generation tool for Solana programs. Built on top of [Codama](https://github.com/codama-idl/codama), Coda provides a zero-config CLI that transforms Anchor IDLs into modern TypeScript clients with full type safety and ES modules support.

## Features

- **Zero Configuration**: Works out of the box with Anchor projects - just run `coda generate`
- **Multiple IDL Support**: Generate clients from multiple programs in one project
- **Full Type Safety**: Generated clients include comprehensive TypeScript types
- **ES Modules**: Modern import/export syntax with `.js` extensions for browser compatibility
- **Account Flattening**: Automatically deduplicates nested account structures from Anchor IDL
- **Custom PDAs**: Define program-derived addresses with custom seeds
- **Codama Visitors**: Apply custom transformations to the AST before generation
- **Rename Support**: Avoid naming conflicts when using multiple IDLs

## Quick Start

Generate your first client:

```bash
# From default Anchor build location (./target/idl/program.json)
bunx @macalinao/coda generate

# From custom IDL location
bunx @macalinao/coda generate --idl ./idls/my_program.json

# Initialize a configuration file
bunx @macalinao/coda init
```

## How It Works

1. **Parse IDL**: Reads your Anchor IDL file (JSON format)
2. **Create AST**: Converts to Codama's node structure
3. **Apply Visitors**: Transforms the AST (custom PDAs, flattening, renaming, etc.)
4. **Generate Code**: Renders TypeScript with ESM support
5. **Output Files**: Creates organized file structure with types, instructions, accounts, etc.

## Generated Client Structure

The generated client provides:
- **Instructions**: Typed builders for all program instructions
- **Accounts**: Decoders and fetchers for all account types
- **Types**: All TypeScript types from the IDL
- **PDAs**: Helper functions for program-derived addresses
- **Errors**: Typed error enums and handlers

## Example Usage

```typescript
import { createTransferInstruction } from "./generated/index.js";
import { createSolanaRpc } from "@solana/web3.js";

const rpc = createSolanaRpc("https://api.mainnet-beta.solana.com");

const instruction = createTransferInstruction({
  source: sourceAddress,
  destination: destAddress,
  authority: authorityAddress,
  amount: 1000n
});
```
